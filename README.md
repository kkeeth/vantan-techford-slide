# Welcome to VANTAN techford academy, javascript class slides

このリポジトリでは、VANTAN techford academyのjavascript初級/中級のスライドを管理しています。

複数のスライドを管理しています。`slides/`フォルダ内の各難易度、各会のスライドを作成してください。

## スライドを起動
`yarn dev ./slides/[授業の難易度]/[回数].md`

## 全てビルド
