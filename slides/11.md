# `Firebase` とは

* 2011年 モバイル・Webアプリケーション開発プラットフォーム
* 2014年 __Google 社に買収__ され [GCP](https://cloud.google.com/) の仲間入り
* `BaaS(Backend as a Service)` がメイン事業
  * 現在は `MBaaS(Mobile BaaS)`
* [AWS Amplify](https://aws.amazon.com) に似たサービス
* 小さなビジネスやプロダクト，個人開発，スタートアップ企業等に勧める
* 様々なサービスや機能を __無料__ で利用可能(※)

※ [ある程度までは無料で，途中から従量課金制に移行](https://firebase.google.com/pricing?hl=ja)

`index.html` の `<body>` タグの終わりに追記する

```html
    </div>

    <!-- ↓↓↓ ここから -->
    <script type="module">
      // Import the functions you need from the SDKs you need
      import { initializeApp } from "https://www.gstatic.com/firebasejs/9.9.3/firebase-app.js";
      // TODO: Add SDKs for Firebase products that you want to use
      // https://firebase.google.com/docs/web/setup#available-libraries

      // Your web app's Firebase configuration
      const firebaseConfig = {
        /* 省略 */
      };

      // Initialize Firebase
      const app = initializeApp(firebaseConfig);
    </script>
    <!-- ↑↑↑ ここまで -->
```
