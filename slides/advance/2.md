# 演習

---

変数の再代入

```javascript
var myName = "Takumi";
myName = "Otobe";

console.log(myName);

let myJob = "UX Engineer"
myJob = "NEET"

console.log(myJob);

const myAge = 23;
myAge = 100;

console.log(myAge);
```

---

再代入は不可だけど定数ではない

