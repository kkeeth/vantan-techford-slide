# 変数の宣言，代入，再代入

```javascript
var myName = "Keeth";
myName = "Kuwahara";

console.log(myName);

let myJob = "UX Engineer"
myJob = "NEET"

console.log(myJob);

const myAge = 23;
myAge = 100;

console.log(myAge);
```

---

# ちょっと余談

再代入は不可だけど定数ではない

```javascript
const sweets = ['パフェ', 'マカロン', 'タルトタタン'];

sweets.push('バナナマフィン');

console.log(sweets);
```

## 配列

```javascript
const numbers = [1, 2, 3, 4, 5]
numbers[0] // 1
```

<br><br>

## オブジェクト

```javascript
const kuwahara = {
  name: 'keeth',
  age: 35
}
kuwahara.age // 35
kuwahara['name'] // keeth
```

# テンプレートリテラル

従来↓

```javascript
const school = 'VANTAN';

console.log("私は" + school + "に通っています")
```

リテラル↓

```javascript
const school = 'VANTAN';

console.log(`私は${school}に通っています`);
```
