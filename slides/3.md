# HTML の構成💁

---

# 大きく4つ

- 📔 `<!DOCTYPE>`:
- 📔 `<html>`: ある意味で HTML 文書の宣言
- 📔 `<head>`: 依存ライブラリやアセットデータ、メタデータなどを配置
- 📔 `<body>`: メインコンテンツ（フローコンテンツと言ったりもする）を配置

---

# もう少し細かく定義

[MDN]() では 7 つのカテゴリに分類されている。

1. メタデータ・コンテンツ
2. フロー・コンテンツ
3. セクショニング・コンテンツ
4. ヘッディング・コンテンツ
5. フレージング・コンテンツ
6. エンベデッド・コンテンツ
7. インタラクティブ・コンテンツ
