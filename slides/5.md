# 第１問💁

<br />

`var`, `let`, `const` の違いについてです.

右のコードを実行するとどうなるでしょう？

::right::

<div class="mt-12">

```js
var foo = "aaa";
foo = 123;  // <- Q1

let bar = "aaa";
bar = 123;  // <- Q2
let bar = 123;  // <- Q3

const baz = "aaa";
baz = 123;  // <- Q4
const baz = 123; // <- Q5

const bar = "aaa"; // <- Q6
let baz = "aaa"; // <- Q7
```

</div>

---
layout: two-cols
---

# 第２問💁

<br />

暗黙の型変換についてです．

右のコードを実行するとどうなるでしょう？


::right::

<div class="mt-12">

```js
123 + "45"; // <- Q1

"123" + 4; // <- Q2

123 - "45"; // <- Q3

"123" - 4; // <- Q4
```

</div>

---
layout: two-cols
---

# 第３問💁

<br />

真偽値についてです．

右のコードを実行するとどうなるでしょう？

::right::

<div class="mt-12">

```js
true && console.log("foo!"); // <- Q1

true || console.log("foo!"); // <- Q2

"" && console.log("foo!"); // <- Q3

0 && console.log("foo!"); // <- Q4

if ({}) console.log("foo!"); // <- Q5
```

</div>

---
layout: two-cols
---

# 第４問💁

<br />

演算子についてです．

右のコードを実行するとどうなるでしょう？

::right::

<div class="mt-12">

```js
let num = 0;
console.log(++num); // <- Q1

console.log(num++); // <- Q2

num = 2;
console.log(num ** 4); // <- Q3
```

</div>
