css でテキストに色を付けてみる．

# CSS の構文

CSS のコードのルールを軽く説明💁

```css
p {
  color: red;
}
```

* セレクタ: `p` の部分
  * セレクタは複数個指定できる
  * ここには HTML 要素以外にも，`class` や `id` も指定できる ※後述します
* 波括弧: `{, }` 内にセレクタのスタイリングルールを記述
* プロパティ: `:` の左側（今回は `color` の部分）
* プロパティの値: `:` の右側（今回は `red` の部分）

※必ず各ルールの最後に `;` を付けること．

---

# CSS の記述を別ファイルに切り出す

<br>

CSS（と JavaScript）ファイルは別途ファイルに切り出し，そこにソースコードを書き，HTML ファイルで読み込むことが一般的．

やってみよう👉

---

`styles.css` というファイルを作成し，先程の CSS の記述をまるっとコピペする．

```css
p {
  color: red;
}
```

そして，`styles.css` を `index.html` ファイルで読み込む．

```diff
  <head>
    <!-- 中略 -->
-   <style>
-     p {
-       color: red;
-     }
-   </style>
+   <link rel="stylesheet" href="style.css">
  </head>
```

# セレクタは複数個指定できる

少し HTML 要素を増やす．

```diff
 <body>
   <h1>Hello CSS</h1>
   <p>CSS基礎</p>
+  <ul>
+    <li>リスト１</li>
+    <li>リスト２</li>
+    <li>リスト３</li>
+  </ul>
 </body>
```

上記で追加したリストの項目にも色をつける．複数セレクタを指定するには __`,` で追記する．__

```css
p, li {
  color: red;
}
```

やってみよう👉

---

# 別々のスタイリングをしたい

それぞれの HTML 要素に個別のスタイルをしたい場合は，分けて記述する．今回はリストの項目の色を緑（`green`）に変更してみる．

```css
p {
  color: red;
}

li {
  color: green;
}
```

やってみよう👉

---

# ルールを重複して設定

<br />

CSS のスタイリングルールは重複して同じセレクタにルールを指定することも可能．

```css
p, li {
  color: red;
}

li {
  color: green;
}
```

その場合，ルールは __後勝ち__ になるので，上記だとリストの項目は __緑色__ になる．

# class 属性
<br />

__HTML 要素の一部（複数可）だけをスタイリングする__ 場合に用いる．以下のように，スタイリングしたい HTML 要素に `class` 属性を付与する．

```diff
  <ul>
    <li>リスト１</li>
-   <li>リスト２</li>
+   <li class="special">リスト２</li>
    <li>リスト３</li>
  </ul>
```

class 属性のスタイリングルールは，以下のように `.(名前)` のセレクタを指定する．

```diff
- li {
+ .special {
+   font-weight: bold;
    color: green;
    list-style-type: circle;
  }
```

---

# HTML 要素と class の複合セレクタ
<br />

スタイリングルールのセレクタは以下のように複合で書くことも可能．

```diff
  <h1>Hello CSS</h1>
- <p>CSS基礎</p>
+ <p class="special">CSS基礎</p>
  <ul>
    <li>リスト１</li>
    <li class="special">リスト２</li>
    <li>リスト３</li>
  </ul>
```

```diff
- .special {
+ li.special {
    font-weight: bold;
    color: green;
    list-style-type: circle;
  }
```
やってみよう👉

---

# ID属性
<br />

`class` 属性と似たもので，同様に HTML 要素の属性に `id` を付与する．id 属性は複数個指定できるが，__原則一つの HTML 内に同じ id 属性は一つのみ__

```diff
  <ul>
    <li>リスト１</li>
-   <li class="special">リスト２</li>
+   <li id="special">リスト２</li>
    <li>リスト３</li>
  </ul>

```

id 属性のスタイリングルールは，以下のように `#(名前)` のセレクタを指定する．

```diff
- li.special {
+ #special {
    font-weight: bold;
    color: green;
    list-style-type: circle;
  }
```

# 色々な結合子

---

# Descendant Combinator(子孫結合子)
<br />

`Descentdant(子孫)` の `Combinator(結合子)`．そのまんまですが（笑）

```diff
  <ul>
    <li>リスト１</li>
    <li id="special">リスト２</li>
-   <li>リスト３</li>
+   <li><em>ここが斜め</em>のリスト３</li>
  </ul>
```

ネストした HTML 要素にスタイリングさせたい場合，以下のように半角スペースを入れてセレクタを指定する．

```diff
- #special {
+ li em {
    font-weight: bold;
    color: green;
    list-style-type: circle;
  }
```

---

# Child Combinator(子結合子)
<br />

こちらもまんま `Child(子)` の `Combinator(結合子)`．以下のように `>` を付けて指定する．

```diff
  <ul>
    <li>リスト１</li>
-   <li id="special">リスト２</li>
+   <li class="special">リスト２</li>
    <li><em>ここが斜め</em>のリスト３</li>
  </ul>
```

```diff
- li em {
+ ul > .special {
  font-weight: bold;
  color: green;
  list-style-type: circle;
}
```

---

# Adjacent Sibling Combinator(隣接兄弟結合子)
<br />

こちらも名前の通り，`Adjacent(隣接)` する `Sibling(兄弟，同じ階層を意味)` の `Combinator(結合子)`．以下のように `+` を付けて指定する．

※ ２つ目の要素が１つ目の *直後* かつ *同じ親を持つ* ことに注意

```diff
/* OK */
- li em {
+ p + ul {
    /* 省略 */
  }
```

```diff
/* NG */
- li em {
+ p + li {
    /* 省略 */
  }
```

---

# General Sibling Combinator(一般兄弟結合子)
<br />

こちらも名前の通り，`General(一般)` の `Sibling(兄弟，同じ階層を意味)` の `Combinator(結合子)`．以下のように `~` を付けて指定する．

※ *１つ目の要素の後* に２つ目のがあり，かつ *同じ親を持つ* ことに注意

```diff
  <p class="special">CSS基礎</p>
+ <div>ほげほげ</div>
  <ul>
    <li>リスト１</li>
    <!-- 省略 -->
  </ul>
```

```diff
/* OK */
- p + ul {
+ p ~ ul {
    /* 省略 */
  }
```

# font-family を指定
<br />

以下のように，`,` 区切りで指定していく．基本的にはサイト全体のフォントは統一で指定し，個別で変えたい場合に都度設定する．

※左から優先される．

```css
body {
  font-family: Josefin sans, Arial, Helvetica, sans-serif;
}
```

オリジナルなフォントを設定することも可能．例として [Google Fonts](https://fonts.google.com/) からテキトーに持ってくる．

```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP&display=swap" rel="stylesheet">
```

```diff
  body {
-   font-family: Josefin sans, Arial, Helvetica, sans-serif;
+   font-family: 'Noto Sans JP', Josefin sans, Arial, Helvetica, sans-serif;
  }
```

---

# font-size を指定
<br />

以下のようにサイズを単位付きや，特定の値で指定する．[詳しくはこちら(MDN)](https://developer.mozilla.org/ja/docs/Web/CSS/font-size)．

```css
p {
  /* 単位 */
  font-size: 16px;
  font-size: 1rem;
  font-size: 1.2em;
  font-size: 15%;

  /* 絶対値 */
  font-size: x-small;
  font-size: small;
  font-size: medium;
  font-size: large;
  font-size: x-large;
}
```

他にもあるので調べてみてください💁

---

# font-weight を指定
<br />

以下のように数字か特定の値で指定する．[詳しくはこちら(MDN)](https://developer.mozilla.org/ja/docs/Web/CSS/font-weight)．

```css
p {
  /* 数値 */
  font-weight: 100;
  font-weight: 200;
  font-weight: 300;
  font-weight: 400;// normal
  font-weight: 500;
  font-weight: 600;
  font-weight: 700;// bold
  font-weight: 800;
  font-weight: 900;

  /* キーワード */
  font-weight: normal;
  font-weight: bold;
}
```
他にもあるので調べてみてください💁

# text-align を指定
<br />

以下のように，キーワードや特殊な値を指定．基本的にはキーワードを指定する．[詳しくはこちら(MDN)](https://developer.mozilla.org/ja/docs/Web/CSS/text-align)．

※書字方向に依存することに注意．

```css
p {
  /* キーワード */
  text-align: start;
  text-align: end;
  text-align: left;
  text-align: right;
  text-align: center;
  text-align: justify;
  text-align: justify-all;
  text-align: match-parent;
}
```

---

# text-decoration を指定
<br />

以下のように，特定の値を並べて指定する．[詳しくはこちら(MDN)](https://developer.mozilla.org/ja/docs/Web/CSS/text-decoration)


```css
p {
  text-decoration: underline;
  text-decoration: overline red;
  text-decoration: wavy overline lime;
}
```

__一括指定プロパティ__ とも呼ばれ，以下の４つのプロパティをまとめて指定することができる．

* `text-decoration-color`
* `text-decoration-line`
* `text-decoration-style`
* `text-decoration-thickness`

---

# line-height を指定
<br />

以下のように，業ボックスの高さをキーワードや数値，単位で指定．[詳しくはこちら(MDN)](https://developer.mozilla.org/ja/docs/Web/CSS/line-height)．

```css
p {
  /* キーワード */
  line-height: normal;

  /* 単位なしの数値．要素のフォントサイズにこちらの値をかける */
  line-height: 3.5;
  /* normal と同じ */
  line-height: 1;

  /* 単位 */
  line-height: 3em;
  line-height: 150%;
  line-height: 32px;
}
```

# margin を指定
<br />

以下のように，上下左右の余白をキーワードや単位で指定する．[詳しくはこちら(MDN)](https://developer.mozilla.org/ja/docs/Web/CSS/margin)．

```css
div {
  /* 四辺すべてに適用 */
  margin: 1em;
  /* 上下 | 左右 */
  margin: 5% auto;
  /* 上 | 左右 | 下 */
  margin: 1em auto 2em;
  /* 上 | 右 | 下 | 左 */
  margin: 2px 1em 0 auto;
}
```

`margin` も一括指定プロパティで，以下の４つのプロパティをまとめて指定することができる．

* `margin-bottom`
* `margin-left`
* `margin-right`
* `margin-top`

<style>
  h1 {
    margin-bottom: 0 !important;
  }
  p {
    margin-top: 0 !important;
  }
</style>

---

# padding を指定
<br />

以下のように，上下左右の余白をキーワードや単位で指定する．[詳しくはこちら(MDN)](https://developer.mozilla.org/ja/docs/Web/CSS/padding)．

```css
div {
  /* 四辺すべてに適用 */
  padding: 0;
  /* 上下 | 左右 */
  padding: 5% 10%;
  /* 上 | 左右 | 下 */
  padding: 1em 2em 2em;
  /* 上 | 右 | 下 | 左 */
  padding: 5px 1em 0 1em;
}
```

`padding` も一括指定プロパティで，以下の４つのプロパティをまとめて指定することができる．

* `padding-bottom`
* `padding-left`
* `padding-right`
* `padding-top`

<style>
  h1 {
    margin-bottom: 0 !important;
  }
  p {
    margin-top: 0 !important;
  }
</style>

---

# background を指定
<br />

背景に色を付けたり，画像を指定したり，その画像の一を調整するためのプロパティ．[詳しくはこちら(MDN)]()．

```css
div {
  /* 背景色を16進数で指定 */
  background: #333;

  /* ボックスと背景色を指定 */
  background: border-box pink;

  /* 背景画像を繰り返し無しで指定 */
  background: no-repeat url("/path/to/image/dir/hoge.png");
}
```

`background` も一括指定プロパティで，他にいくつかのプロパティが存在する💁

---

# width, height を指定
<br />

HTML 要素の `width(横幅)`, `height(高さ)` をキーワードや単位で指定する．

```css
div {
  /* 単位 */
  width: 500px;
  width: 30%;
  height: 300px;
  /* 計算もできる */
  height: calc(50% - 40px);

  /* キーワード */
  width: 100vw;
  height: 100vh;
}
```

これ以外にも指定の仕方があるので調べてみてください．
