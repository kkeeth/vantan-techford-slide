# CSS とは

CSS(Cascading StyleSheet) の略で，__HTML コンテンツの装飾を担う__ もの．

HTML 同様プログラミング言語 __ではない__．

しかし CSS 職人達でこんなものを作る人もいる…（ヤバい）

## 兎にも角にも書いてみよう 🙋

css でテキストに色を付けてみる．

---

`index.html` というファイルを作成し，以下を `<body>` タグ内， `<head>` タグの最後にそれぞれ追記．

__index.html__

```html
<body>
  <h1>Hello CSS</h1>
  <p>CSS基礎</p>
</body>
```

```html
<head>
  <!-- 中略 -->
  <style>
    p {
      color: red;
    }
  </style>
</head>
```

---

# CSS の構文

CSS のコードのルールを軽く説明💁

```css
p {
  color: red;
}
```

* セレクタ: `p` の部分
  * セレクタは複数個指定できる
  * ここには HTML 要素以外にも，`class` や `id` も指定できる ※後述します
* 波括弧: `{, }` 内にセレクタのスタイリングルールを記述
* プロパティ: `:` の左側（今回は `color` の部分）
* プロパティの値: `:` の右側（今回は `red` の部分）

※必ず各ルールの最後に `;` を付けること．

---

# CSS の記述を別ファイルに切り出す

<br>

CSS（と JavaScript）ファイルは別途ファイルに切り出し，そこにソースコードを書き，HTML ファイルで読み込むことが一般的．

やってみよう👉

---

`styles.css` というファイルを作成し，先程の CSS の記述をまるっとコピペする．

```css
p {
  color: red;
}
```

そして，`styles.css` を `index.html` ファイルで読み込む．

```diff
  <head>
    <!-- 中略 -->
-   <style>
-     p {
-       color: red;
-     }
-   </style>
+   <link rel="stylesheet" href="style.css">
  </head>
```

# セレクタは複数個指定できる

少し HTML 要素を増やす．

```diff
 <body>
   <h1>Hello CSS</h1>
   <p>CSS基礎</p>
+  <ul>
+    <li>リスト１</li>
+    <li>リスト２</li>
+    <li>リスト３</li>
+  </ul>
 </body>
```

上記で追加したリストの項目にも色をつける．複数セレクタを指定するには __`,` で追記する．__

```css
p, li {
  color: red;
}
```

やってみよう👉

---

# 別々のスタイリングをしたい

それぞれの HTML 要素に個別のスタイルをしたい場合は，分けて記述する．今回はリストの項目の色を緑（`green`）に変更してみる．

```css
p {
  color: red;
}

li {
  color: green;
}
```

やってみよう👉

---

# ルールを重複して設定

<br />

CSS のスタイリングルールは重複して同じセレクタにルールを指定することも可能．

```css
p, li {
  color: red;
}

li {
  color: green;
}
```

その場合，ルールは __後勝ち__ になるので，上記だとリストの項目は __緑色__ になる．

# class 属性
<br />

__HTML 要素の一部（複数可）だけをスタイリングする__ 場合に用いる．以下のように，スタイリングしたい HTML 要素に `class` 属性を付与する．

```diff
  <ul>
    <li>リスト１</li>
-   <li>リスト２</li>
+   <li class="special">リスト２</li>
    <li>リスト３</li>
  </ul>
```

class 属性のスタイリングルールは，以下のように `.(名前)` のセレクタを指定する．

```diff
- li {
+ .special {
+   font-weight: bold;
    color: green;
    list-style-type: circle;
  }
```

---

# HTML 要素と class の複合セレクタ
<br />

スタイリングルールのセレクタは以下のように複合で書くことも可能．

```diff
  <h1>Hello CSS</h1>
- <p>CSS基礎</p>
+ <p class="special">CSS基礎</p>
  <ul>
    <li>リスト１</li>
    <li class="special">リスト２</li>
    <li>リスト３</li>
  </ul>
```

```diff
- .special {
+ li.special {
    font-weight: bold;
    color: green;
    list-style-type: circle;
  }
```
やってみよう👉

---

# ID属性
<br />

`class` 属性と似たもので，同様に HTML 要素の属性に `id` を付与する．id 属性は複数個指定できるが，__原則一つの HTML 内に同じ id 属性は一つのみ__

```diff
  <ul>
    <li>リスト１</li>
-   <li class="special">リスト２</li>
+   <li id="special">リスト２</li>
    <li>リスト３</li>
  </ul>

```

id 属性のスタイリングルールは，以下のように `#(名前)` のセレクタを指定する．

```diff
- li.special {
+ #special {
    font-weight: bold;
    color: green;
    list-style-type: circle;
  }
```

# Descendant Combinator(子孫結合子)
<br />

`Descentdant(子孫)` の `Combinator(結合子)`．そのまんまですが（笑）

```diff
  <ul>
    <li>リスト１</li>
    <li id="special">リスト２</li>
-   <li>リスト３</li>
+   <li><em>ここが斜め</em>のリスト３</li>
  </ul>
```

ネストした HTML 要素にスタイリングさせたい場合，以下のように半角スペースを入れてセレクタを指定する．

```diff
- #special {
+ li em {
    font-weight: bold;
    color: green;
    list-style-type: circle;
  }
```

---

# Child Combinator(子結合子)
<br />

こちらもまんま `Child(子)` の `Combinator(結合子)`．以下のように `>` を付けて指定する．

```diff
  <ul>
    <li>リスト１</li>
-   <li id="special">リスト２</li>
+   <li class="special">リスト２</li>
    <li><em>ここが斜め</em>のリスト３</li>
  </ul>
```

```diff
- li em {
+ ul > .special {
  font-weight: bold;
  color: green;
  list-style-type: circle;
}
```

---

# Adjacent Sibling Combinator(隣接兄弟結合子)
<br />

こちらも名前の通り，`Adjacent(隣接)` する `Sibling(兄弟，同じ階層を意味)` の `Combinator(結合子)`．以下のように `+` を付けて指定する．

※ ２つ目の要素が１つ目の *直後* かつ *同じ親を持つ* ことに注意

```diff
/* OK */
- li em {
+ p + ul {
    /* 省略 */
  }
```

```diff
/* NG */
- li em {
+ p + li {
    /* 省略 */
  }
```

---

# General Sibling Combinator(一般兄弟結合子)
<br />

こちらも名前の通り，`General(一般)` の `Sibling(兄弟，同じ階層を意味)` の `Combinator(結合子)`．以下のように `~` を付けて指定する．

※ *１つ目の要素の後* に２つ目のがあり，かつ *同じ親を持つ* ことに注意

```diff
  <p class="special">CSS基礎</p>
+ <div>ほげほげ</div>
  <ul>
    <li>リスト１</li>
    <!-- 省略 -->
  </ul>
```

```diff
/* OK */
- p + ul {
+ p ~ ul {
    /* 省略 */
  }
```
