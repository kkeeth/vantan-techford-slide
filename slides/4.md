# 兎にも角にも書いてみよう 🙋

css でテキストに色を付けてみる．

---

`index.html` というファイルを作成し，以下を `<body>` タグ内， `<head>` タグの最後にそれぞれ追記．

__index.html__

```html
<body>
  <h1>Hello CSS</h1>
  <p>CSS基礎</p>
</body>
```

```html
<head>
  <!-- 中略 -->
  <style>
    p {
      color: red;
    }
  </style>
</head>
```

# CSS の構文

CSS のコードのルールを軽く説明💁

```css
p {
  color: red;
}
```

* セレクタ: `p` の部分
  * セレクタは複数個指定できる
  * ここには HTML 要素以外にも，`class` や `id` も指定できる ※後述します
* 波括弧: `{, }` 内にセレクタのスタイリングルールを記述
* プロパティ: `:` の左側（今回は `color` の部分）
* プロパティの値: `:` の右側（今回は `red` の部分）

※必ず各ルールの最後に `;` を付けること．

---

# CSS の記述を別ファイルに切り出す

<br>

CSS（と JavaScript）ファイルは別途ファイルに切り出し，そこにソースコードを書き，HTML ファイルで読み込むことが一般的．

やってみよう👉

---

`styles.css` というファイルを作成し，先程の CSS の記述をまるっとコピペする．

```css
p {
  color: red;
}
```

そして，`styles.css` を `index.html` ファイルで読み込む．

```diff
  <head>
    <!-- 中略 -->
-   <style>
-     p {
-       color: red;
-     }
-   </style>
+   <link rel="stylesheet" href="style.css">
  </head>
```

# セレクタは複数個指定できる

少し HTML 要素を増やす．

```diff
 <body>
   <h1>Hello CSS</h1>
   <p>CSS基礎</p>
+  <ul>
+    <li>リスト１</li>
+    <li>リスト２</li>
+    <li>リスト３</li>
+  </ul>
 </body>
```

上記で追加したリストの項目にも色をつける．複数セレクタを指定するには __`,` で追記する．__

```css
p, li {
  color: red;
}
```

やってみよう👉

---

# 別々のスタイリングをしたい

それぞれの HTML 要素に個別のスタイルをしたい場合は，分けて記述する．今回はリストの項目の色を緑（`green`）に変更してみる．

```css
p {
  color: red;
}

li {
  color: green;
}
```

やってみよう👉

---

# ルールを重複して設定

CSS のスタイリングルールは重複して同じセレクタにルールを指定することも可能．

```css
p, li {
  color: red;
}

li {
  color: green;
}
```

その場合，ルールは __後勝ち__ になるので，上記だとリストの項目は __緑色__ になる．

# デフォルトのスタイリングを上書き

上書きしてみよう．今回はリストの頭のドット（`・`）を変更してみる．

```diff
  li {
    color: green;
+   list-style-type: circle;
  }
```
他にも色んな設定があるので，やってみよう👉

## コメント

CSS もコメントを書くことができる．

```css
  li {
    color: green;
    list-style-type: circle;
    /* ここはコメント */
  }

```
